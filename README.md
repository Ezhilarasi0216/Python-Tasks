# Python If Condition Programs

## Project Description
This repository contains simple Python programs that demonstrate the use of **if, elif, and else conditions**. These programs help beginners understand decision-making statements in Python.

---

## Topics Covered
- Basic if condition
- if-else statement
- if-elif-else ladder
- Nested if statements
- Logical conditions

---

## Example Program

### Check Even or Odd Number

```python
num = int(input("Enter a number: "))

if num % 2 == 0:
    print("The number is Even")
else:
    print("The number is Odd")
